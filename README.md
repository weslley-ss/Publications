# Publications
This repository serves as a long-term archive of my academic publications and research materials. It contains peer-reviewed articles, preprints, extended abstracts, posters, and presentation slides developed throughout my academic career. The goal of this repository is to maintain an open and organized record of my scientific contributions, ensuring accessibility, transparency, and reproducibility of the research.

Where applicable, each publication is accompanied by links to datasets, code implementations, or supplemental documents that support the results presented. When references to proprietary or confidential data are involved, only public or derivative materials are included.

This repository is continuously updated as new research is developed and published.
